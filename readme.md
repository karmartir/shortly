# Shortly

## Project Overview
Shortly is a static landing page and demo project that showcases a clean and modern design for a URL shortening service. It demonstrates the visual layout and user interface elements without backend functionality.

## Features
- Clean and responsive landing page design
- Demo layout for URL shortening interface
- User-friendly and intuitive UI components
- Visual representation of a URL shortening service

## Tech Stack
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
